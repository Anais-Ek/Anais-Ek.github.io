ANAMANA (en cours)

Dans le cadre de nos études, nous devons réaliser un projet de fin d’année pour notre examen de BTS. Avec mes camarades Nathalie LIU et Marcel Cavaretta, nous avons choisi le robot Orientation boutique que nous avons décidé de nommer ANAMANA (association des premières lettres de nos prénoms).

Le but du projet est de développer un agent conversationnel, sous forme de borne, pour accueillir les clients dans un magasin et les orienter vers les différents rayons et produits. Nous avons choisi de faire une boutique de vêtements mixte avec sur les produits notre logo.

Nous devons créer une interface de configuration permettant aux managers des rayons de lister les nouveaux produits, leur emplacement, et changer éventuellement l’emplacement des produits ou les supprimer des rayons. Le système devra produire des statistiques sur les produits les plus demandés ou les produits en rupture de stock.

L’agent conversationnel sera implémenté sur une tablette raspberry muni d’un microphone et des enceintes et comprendra les technologies suivantes :

  • Reconnaissance Vocale
  • Synthèse de la parole
  • Compréhension de la parole

Nous allons également développer une autre interface pour le client qui va afficher ce qu’il a demandé sous forme de messagerie. Voici un exemple fait sur figma :

Il sera relié à une mini-imprimante pour imprimer les indications au client pour s’orienter vers le produit qu’il cherche.

L’implémentation sera faite dans un font-end web, et utilisera les moteurs de reconnaissance vocale et de synthèse de la parole de Chrome. Le module de compréhension de la demande et d’identification du nom du produit sera développé dans un backend (NodeJS) et sera relié à une base de données (MySQL).

Les langages de programmations que nous allons utiliser :

  • FrontEnd : HTML / CSS / Javascript
  • BackEnd : NodeJS
  • Base de données : MySQL
